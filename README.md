## 🌟 Introduction

Welcome to Rabbit Tree Hostel Management System!  
This is a modern, mobile-friendly, bilingual (English & Indonesian) web app for hostel operations—**assign rooms, track cleaning, gamify staff performance, and keep managers and owners fully in control**.

Built for teams of up to **15 staff** and **10 supervisors**, with full analytics, spot-check workflow, and robust offline capabilities. All real cleaning tasks and room data are included.  
No code download needed—just use the live app!

### 2. **How Each User Type Uses the System**

#### 👤 **Staff**
- Log in (`staff1`–`staff15`, password: `staff123`).
- See your room assignments for the day.
- Mark off each cleaning task as you go (tasks are all real, bilingual, and track your progress).
- Get badges and streaks for a job well done.

#### 👁️‍🗨️ **Supervisor**
- Log in (`super1`–`super10`, password: `super123`).
- See all completed rooms and perform spot checks (one per day per supervisor).
- Pass or fail rooms; leave notes if improvements are needed.
- Failed rooms are flagged for staff to re-clean before the day ends.

#### 📊 **Manager**
- Log in (`manager1`, password: `manager123`).
- Access the analytics dashboard to see performance, cleaning trends, and exports.
- Managers can’t assign rooms or users (owner only).

#### 👑 **Owner**
- Log in as above.
- Manage all users, room setups, assignments, and settings.
- Monitor system health, view all analytics, and get storage/export alerts.
- Export data (CSV/JSON), trigger room assignments, and configure retention/archiving.

## 📲 System Features (At a Glance)

- **453 real cleaning tasks** across 15 rooms, in EN/ID, with natural language and typo fixes
- **11 unique cleaning sections** (entry, bedding, facilities, toilet, terrace, and more)
- **4 roles:** Staff, Supervisor, Manager, Owner (each with unique UI & permissions)
- **Gamification:** Badges for daily streaks, room milestones, quality passes
- **Spot checks:** 1 per supervisor per day, with notes and fail/pass actions
- **Real-time analytics:** Performance, quality, completion rates, exportable as CSV/JSON
- **Offline support:** Works with no internet after first load
- **Data retention & archiving:** 90-day minimum, with export reminders when storage fills
- **Per-user language switching:** Remembers each user's choice (EN/ID)
- **Mobile-first, responsive design:** Works on phones, tablets, desktops

## 📝 Usage Notes

- **Add more users** in User Management (owner only).
- **Assign rooms** daily—system auto-assigns up to 5 rooms/staff in round-robin.
- **Spot check workflow:** Failed rooms are flagged “Needs Attention”—staff can’t complete day until rechecked.
- **Switch language:** Use the top-right toggle; your choice is saved for next time.
- **Data export:** Owner/Manager can export history as CSV or JSON for backups or reporting.
- **Mobile shortcut:** Add the app to your home screen for true offline PWA experience.

---

## 💡 Pro Tips for New Owners

- After setup, give each team member their login for direct, accountable usage.
- Encourage staff to check off tasks in real time for better analytics.
- Supervisors should spot check different staff daily for fair quality control.
- Use exports to backup your data regularly, especially before clearing browser storage or devices.
- If storage is nearly full, system will prompt for data export/archiving—don’t ignore it!
